Practica

Esta es una práctica de algunas etiquetas de HTML5 y CSS



En la siguiente tabla podemos observar algunos elementos de las etiquetas y para qué se utilizan:

Elemento Función
P Agregar texto
IMG Agregar una imagen
UL Agregar una lista numerada
OL Agregar una lista sin numeración
TABLE Agregar una tabla
THEAD Agregar el encabezado de la tabla
TBODY Agregar el contenido de la tabla
BR Colocar el siguiente texto en otro renglón
STRONG Colocar en negrita una palabra o texto
EM Colocar en cursiva una palbra o texto
A Colocar un vínculo
DIV Seleccionar varios elementos,
como por ejemplo varios párrafos.
Permite aplicar una misma regla de CSS a varios elementos
(los que abarca la etiqueta DIV )
SPAN Permite seleccionar una palabra u oración dentro de un párrafo
para luego aplicarle una regla de CSS.

En la siguiente tabla podemos observar algunas propiedades de CSS y para qué se utilizan:

Propiedad Función
COLOR Definir un color para el texto
BACKGROUND-COLOR Agregar un color de fondo a un texto o a un conjunto de elementos
FONT-FAMILY Permite seleccionar el tipo de letra que vamos a utilizar y configurar por ejemplo el grosor, estilo
(Ejemplo: Helvética, Cursive, Monospace, Serif, Sans-serif, etc).
Son como los diferentes tipos de letras (o fuentes) que podemos usar.
FONT-SIZE Permite cambiar el tamaño de la letra.
Se establece una regla en CSS con el elemento sobre el cual se desea aplicar la propiedad.
Dentro de las unidades de medida que se utilizan están los Pixeles (px) y se coloca el número seguido de px (Ejemplo: 100px).
Otra unidad de medida es EM, que establece un valor relativo de aumento o disminución (Por ejemplo: 2em).
También se encuentran las unidades EX y % que definen un aumento relativo del texto o palabra a modificar.
FONT-STYLE Permite colocar un estilo Cursiva a la fuente seleccionada en font-family.
Los valores que se usan en font-style son:
  1. Normal : mantiene original el estilo de la fuente seleccionada.
  2. Italic : coloca un estilo de letra cursiva.
  3. Oblique : coloca un estilo diferente de cursiva.
FONT-WEIGHT Permite darle un estilo Negrita al texto. Se utiliza el valor BOLD .
Tanto FONT-WEIGHT como FONT-STYLE generan el mismo efecto que las etiquetas strong y em, respectivamente,
la diferencia radica en que las propiedades aplican mejor cuando son párrafos,
mientras que las etiquetas mencionadas se adecuan más a palabras o frases.
LINE-HEIGHT Definir el espacio entre líneas (interlineado) en un párrafo.
Los valores se pueden expresar en porcentajes, números, un valor seguido de em .
Cuando se utiliza el porcentaje o em , el espacio es calculado de acuerdo al tamaño del texto;
TEXT ALIGN Permite alinear un texto, imagen o ambos .
Los valores que se utilizan son los siguientes:
  1. Left : Alinea el contenido a la izquierda.
  2. Right : Alinea el contenido a la derecha.
  3. Center : Permite centrar el contenido.
  4. Justify : Permite justificar el contenido.
MARGIN Permite alinear una tabla . Por ser la tabla un elemento en bloque, la propiedad Text-align no aplica.
Uno de los valores a utilizar sería auto .
TEXT-DECORATION Permite subrayar un texto o un párrafo. También permite colocar la línea encima del texto y
colocar la línea en el texto dando la apariencia de tachado.
Los valores que se utilizan en la propiedad Text-decoration son los siguientes:
  1. Underline : Permite subrayar un texto
  2. Overline : Agrega la línea en la parte superior del texto.
  3. Line-through : Permite tachar el texto.
WIDTH Permite definir la anchura de un elemento dentro de las etiquetas div.
Puede Expresarse en "px" o en "%".
HEIGHT Permite definir la altura de un elemento dentro de las etiquetas div.
Puede Expresarse en "px" o en "%".
OVERFLOW Permite modificar la presentación del contenido de un cuadro cuando a este último
se le ha modificado su altura o anchura y el contenido aparece desbordado.
Cuando se utiliza una etiqueta div los elementos contenidos en ella hacen parte de un bloque
y pueden modificarse o desplazarse en la página. En ese sentido cuando reducimos la anchura o altura
de un bloque, el contenido o el texto que hay dentro del cuadro puede aparecer desbordado (y sobrepuesto al
contenido adyacente) ya que se reducen las dimensiones del cuadro, que lo se puede corregir con la propiedad
Overflow, que permite colocar barras de desplazamiento o cortar el texto para que no se vea desbordado

Los valores que aplican en la propiedad Overflow son:
  1. Visible: es el predeterminado y se verá desbordado si se reduce el tamaño del cuadro.
  2. Hidden: permite cortar el texto hasta donde llegue el cuadro.
  3. Auto: permite agregar barras de desplazamiento al cuadro (y no corta el texto). Si se desea
    colocar sobre el eje de las "Y" se escribe "Overflow-y", y si se desea colocar
    colocar sobre el eje de las "X" se escribe "Overflow-x".
